Manage API Keys

Use API keys to authenticate requests to the Llama API. This guide explains how to create and revoke API keys safely.

Only Platform Administrators and Platform Owners can manage API keys.

Create an API Key

You can create up to 10 API keys. Each key has the same access rights as your account.

  1. Go to your Admin account.
  2. In the Platform settings section, click API.
  3. Click Add API key.
  4. When asked, enter a meaningful label for the API key. For example, indicate which integration uses this API key.
  5. Click Create.
  6. Copy your API key and store it securely.
You can only view the API key once. If you lose it, you must create a new one.
  1. Click Close.

Revoke an API key

Revoke an API key if it is no longer needed or if you suspect it has been compromised.

Revoking an API key permanently deletes it. Any integration using this key will stop working immediately.

  1. Go to your Admin account.
  2. In the Platform settings section, click API.
  3. Find the API key you want to revoke, then click Revoke.
  4. When asked, click Yes, revoke.
