Manage API Keys

The Llama API authenticates requests using API Keys. These API keys represent the required credentials to use the Llama API.

Only Platform Administrators and Platform Owners can manage API keys.

Create an API Key

From your Admin account, you can create up to 10 API keys that carry the same access rights for use with the platform.

  1. Go to your Admin account.
  2. In the Platform settings section, click API.
  3. Click Add API key.
  4. When asked, enter a meaningful label for the API key. For example, indicate which integration uses this API key.
  5. Click Create.
  6. Copy your API key somewhere safe.

    You can only view the API key once. If you lose this API key, you need to create a new one.

  7. Click Close.

Revoke an API key

If you no longer use an API or think an API key has been compromised, you can revoke it.

Once an API key is revoked, it is permanently deleted. Any associated integration stops working.

  1. Go to your Admin account.
  2. In the Platform settings section, click API.
  3. Find the API key you want to revoke, then click Revoke.
  4. When asked, click Yes, revoke.
